# 🧰 Ultimate macOS CLI Environment

**Автоматизируйте настройку вашего Mac для максимальной продуктивности в разработке и DevOps.** Единая команда разворачивает мощную, готовую к работе среду с нуля.

[![Quick Start](https://img.shields.io/badge/Quick-Start-brightgreen)](#-быстрый-старт) 
![Homebrew](https://img.shields.io/badge/Powered_by-Homebrew-orange)

Этот репозиторий содержит тщательно подобранный `Brewfile`, который превращает чистую систему macOS в полноценную рабочую станцию с инструментами для CLI, облачных технологий, безопасности и мультимедиа.

---

## ✨ Ключевые возможности

| Область | Основные инструменты |
| :--- | :--- |
| **🎯 Продуктивность в CLI** | `zsh` + `powerlevel10k`, `fzf`, `eza`, `bat`, `zoxide`, `yazi` |

## 🚀 Быстрый старт

Настройте всю среду за несколько минут.

### 1. Установите Homebrew (если не установлен)
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```


### 💻 Командная строка и инструменты
Шелл и тема: Современный zsh с красивым и функциональным оформлением powerlevel10k

Навигация: zoxide для умного перехода по папкам, yazi — быстрый файловый менеджер на терминал

Визуализация: eza (улучшенный ls), bat (улучшенный cat), btop (мониторинг ресурсов)

Поиск: fzf для нечеткого поиска, ripgrep и fd для быстрого поиска файлов и кода

Редактирование: neovim как мощный редактор с готовыми зависимостями

### ☁️ Kubernetes и DevOps
Полный стек для работы с Kubernetes: kubectl, helm, argocd

k9s — терминальный дашборд для удобного управления кластерами

Вспомогательные утилиты: docker, jq для работы с JSON

### 🔐 Безопасность и Сеть
Криптография: gnupg, openssl@3

Анализ трафика: wireshark (CLI-компоненты)

Библиотеки: libsodium, zeromq, libssh

### 🖼️ Мультимедиа и Обработка
Обработка видео/аудио: ffmpeg (с кодеками x264, x265), flac, mpg123

Работа с изображениями: imagemagick для конвертации, tesseract для OCR

### ⚙️ Системные библиотеки и зависимости
Важные системные библиотеки: ncurses, readline, libyaml, sqlite, zstd, brotli и другие

### 🎨 GUI и шрифты
colorwell — утилита для подбора цветов

Symbols Nerd Font — шрифт с иконками для корректного отображения в терминале

### 🛠️ Использование
После установки все инструменты доступны из командной строки. Вот некоторые полезные команды:

Настройте powerlevel10k, если это ваш первый запуск (скрипт проведет вас через процесс)

Убедитесь, что в вашем терминале (например, iTerm2) выбран шрифт "Symbols Nerd Font" для корректного отображения иконок

### 🔄 Обновление
Для обновления всех установленных пакетов выполните:

```bash
brew update && brew upgrade
```
